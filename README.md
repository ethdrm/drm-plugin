## Description  
Plugin for IntelliJ IDEA. Showcase for ethdrm smart contract.
  
## Requirements  
* IntelliJ IDEA (2016.2 and higher)
* Java 8
* Gradle  

## Usage  
Simply run 'gradle runIde' in project directory.
